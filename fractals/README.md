Ce projet consiste à dessiner une fractale à partir de n'importe quel dessin proposé. Le motif dessiné est composé de plusieurs points en forme du motif, eux-mêmes composés de points en forme du motif. On dessine en restant appuyé avec le curseur de la souris et on peut effacer le dessin en cliquant sur n'importe quelle touche.
Ce code s'exécute sur Processing.
